URL TO THE MAIN PAGE: Main Page
  1. First, you need index.html and main.css ready.
  2. Add html, nav, header, etc. Remember, always end with a slash.
  3. Add the text you want inbetween < > and < / >.
  4. Make sure you have !doctype html and link rel="stylesheet" href="main.css" type="text/css" on top of your index file.
  5. For CSS, it is another document that makes your index file look better on the webpage.
  6. After typing a command in CSS, always end with ;
  7. To save, use git add first, then do git commit -m, last, git push.
  8. For sftp, enter username, go in public_html, and put .
  9. Use get to download a file.
